# Legis CPM

Lector de legislaci&oacute;n de Conservatorios Profesionales de M&uacute;sica de la Generalitat Valenciana.

Aplicaci&oacute;n web est&aacute;tica que permite consultar, buscar y analizar la normativa vigente que regula los conservatorios profesionales de m&uacute;sica en la Comunitat Valenciana. Disponible en castellano y valenciano.

## Caracter&iacute;sticas

- Cat&aacute;logo de legislaci&oacute;n organizado por categor&iacute;as
- Buscador global (por nombre de ley o t&eacute;rmino en el texto)
- Lector de legislaci&oacute;n con &iacute;ndice lateral navegable
- Sistema de versiones de art&iacute;culos modificados
- An&aacute;lisis jur&iacute;dico (vigencia, afectaciones, conformidad)
- Biling&uuml;e: castellano y valenciano
- Descarga en JSON y Markdown

## Tecnolog&iacute;a

- [Astro](https://astro.build/) &mdash; Generador de sitios est&aacute;ticos
- GitHub Pages &mdash; Hosting
- FlexSearch &mdash; B&uacute;squeda del lado del cliente
